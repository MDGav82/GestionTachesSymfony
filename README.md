# GestionTachesSymfony
Projet PHP utilisant Symphony et Twig

# Mise en place
1. (CMD) git clone
2. (CODE) changer le DATABASE_URL du .env
3. (CMD) composer install
4. (CMD) symfony serve
5. (URL) localhost/login
